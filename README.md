# 玻璃瓶检测与瓶口定位

本项目为《计算机视觉导论》期末课程项目，包括玻璃瓶检测和瓶口定位两个任务。

## 文件说明

- `GlassImg`为数据存放地址，包含`Circle`, `Ellipse`, `Raindrop`和`mix`四个数据集，每个数据集按照4:1划分训练集和测试集，数据格式为`yolo`格式。

- `models`为模型存放地址，包含yolo模型代码以及模型参数。
  - `runs`储存了训练、测试和检测的过程文件
  - `models`储存了模型结构配置文件
  - `data`储存了加载数据的配置文件
  - `run.sh`为执行训练、测试、检测的脚本
